# Guess The Number 🎮

A terminal-based number guessing game built in Python.

## Features
- 5 difficulty levels (1-10 up to 1-500)
- Attempt limit per level
- Score system based on remaining attempts
- Colorful terminal UI with Colorama
- Random bot messages for a dynamic experience
- Replay option after each game

## Requirements
pip install colorama

## How to play
Run the script and follow the instructions in the terminal.
Choose your difficulty level and try to find the bot's secret number
before running out of attempts !
